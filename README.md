🎨 Anima WebUI <small>v1.2.3</small>

Anima WebUI は、ComfyUI をバックエンドとして利用し、直感的な操作で高品質な画像生成を行うための Gradio ベースの Web UI です。 v1.2.3 では、自由なタグ管理と、さらに洗練された履歴復元機能を提供します。
🎯 想定するユーザ層

    ComfyUI を既に導入済み、あるいは自力でインストールできる方

    必要なモデルを自身でダウンロードし、所定のフォルダへ配置できる方

    技術的な前提条件を理解し、環境構築の「罠」を自力で回避できる知性をお持ちの方

🛠 事前準備
1. ソフトウェア環境

    Python 3.12.x: インストール時に必ず [Add Python to PATH] にチェックを入れてください。

    ComfyUI: 画像生成のバックエンドとして動作している必要があります。

2. モデル & ワークフローの導入

Hugging Face 等から以下のファイルを配置してください。

    モデルファイル: models/checkpoints/ フォルダへ配置。

    ワークフローファイル: anima-t2i.json を本アプリのルートフォルダに配置。

🚀 セットアップと起動手順

    リポジトリの準備
    PowerShell

    git clone https://github.com/okpn/anima-webui.git
    cd anima-webui

    設定ファイルの簡素化

        config.json.sample をコピーして config.json を作成します。

        v1.2.3 では、app_name や version などの固定値は config_utils.py が管理するため、JSON 側からは削除して「自分が変更したい項目だけ」を記述するミニマル運用が可能です。

    起動

        start_anima_webui.bat を実行します。

💡 主な機能 (v1.2.3)
タブ名	機能概要
Generate	

Custom Tags (My Palette): 【新機能】ユーザーが考案した独自のタグセットを表示。自分だけの「絵作りのスパイス」をワンタップで追加可能。


翻訳アコーディオン: 垂直スペースを削減した DeepL プロンプト・ブリッジ。


年代スロット: 最大 3 つの年代を混合指定できるブレンディング機能。
History	

全 22 パラメータ復元: 新たに Custom Tags を含めた、生成時の全パラメータを完璧に復元。過去の「奇跡の 1 枚」をいつでも再現できます。


URL 自動修復: スマホ操作時のパス問題を解決するクリーンアップ機能。
System	

タグ・エディタ: Custom Tags を含む各カテゴリのタグリストを UI 上で直接編集・保存可能。


IP 自動検出: サーバーのローカル IP を検出し、スマホからのアクセスを容易にします。
⚠️ 注意事項

    設定の保護: API キーを含む config.json の取り扱いには注意してください。

    Windows の「譲歩」を活用: 本アプリは、Windows が AI ソフトウェアのために用意したメモリ退避路やリソース管理を最大限に活用するように設計されています。

    「被害者」を出さない設計: 二重起動防止やパスの自動修正により、ComfyUI 特有の「罠」をユーザーに意識させません。
